# Healthcare CRM Frontend

Frontend for the Healthcare CRM system built using React, Tailwind CSS, Axios, and React Router.

## Features

- Responsive dashboard UI
- Patients management
- Doctors management
- Appointment management
- Status-based appointment actions
- Search and filtering
- Active/inactive handling

## Tech Stack

- React
- Tailwind CSS
- Axios
- React Router
- Vite

## Run Locally

```bash
npm install
npm run dev
````

Frontend runs on:

```txt
http://localhost:5173
```
