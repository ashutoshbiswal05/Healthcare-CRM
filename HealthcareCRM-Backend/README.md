# Healthcare CRM Backend

Backend REST API for the Healthcare CRM system built using Spring Boot and MySQL.

## Features

- Patients API
- Doctors API
- Appointments API
- Appointment lifecycle management
- Soft delete handling
- Status-based restrictions
- Case-insensitive enum updates

## Tech Stack

- Java
- Spring Boot
- Hibernate
- MySQL
- Gradle

## Run Locally

```bash
./gradlew bootRun

Backend runs on:

http://localhost:8080
